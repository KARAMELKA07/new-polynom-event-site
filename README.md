## О проекте

Проект представляет собой одностраничный React-сайт, разработанный для сопровождения отчётного концерта вокальной студии.

Главная интерактивная часть сайта — карта героев спектакля. Пользователь может перемещать и масштабировать карту, открывать карточки персонажей, изучать их описание и связи, а затем проголосовать за предполагаемого виновника.

Интерфейс адаптирован для компьютеров и мобильных устройств. Для управления картой поддерживаются мышь, сенсорное перемещение и жест масштабирования двумя пальцами.

## Основные возможности

- адаптивная одностраничная вёрстка;
- навигация по разделам страницы;
- интерактивная карта героев спектакля;
- перемещение карты с помощью мыши и touch-жестов;
- масштабирование колёсиком мыши, кнопками и жестом pinch-to-zoom;
- автоматическая адаптация карты под размер экрана;
- кнопка возврата к исходному положению карты;
- модальные карточки с описанием героев;
- закрытие модального окна по клавише `Escape`;
- предварительная загрузка изображений персонажей;
- опрос зрителей через Яндекс Формы;
- форма для сбора отзывов;
- ссылки на социальные сети вокальной студии;
- базовые атрибуты доступности для интерактивных элементов.

## Интерфейс

### Интерактивная карта персонажей

Карта построена на основе данных о героях и позволяет перемещаться между персонажами, изучать их связи и открывать подробные карточки.

### Карточки героев

Для каждого персонажа предусмотрены фотография, роль в сюжете, краткое описание и характерные особенности.

## Компоненты

### `FamilyTreeCanvas`

Отвечает за отображение и управление интерактивной картой:

- перемещение карты;
- масштабирование;
- touch-жесты;
- выбор персонажа;
- адаптацию под размер экрана;
- предварительную загрузку изображений.

### `CharacterModal`

Отображает подробную информацию о выбранном персонаже:

- фотографию;
- имя и роль;
- описание;
- характерные особенности;
- закрытие по кнопке и клавише `Escape`.

### `SuspectPoll`

Показывает приглашение принять участие в расследовании и открывает опрос в Яндекс Формах.

### `familyTree.js`

Содержит данные проекта:

- тексты основных разделов;
- информацию о персонажах;
- координаты элементов карты;
- изображения;
- настройки опроса.

## Локальный запуск

### Требования

- Node.js 18 или новее;
- npm.

### Установка

```bash
git clone https://github.com/KARAMELKA07/new-polynom-event-site.git
cd new-polynom-event-site
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

После запуска приложение будет доступно по адресу, который Vite покажет в терминале.

### Production-сборка

```bash
npm run build
```

Готовая сборка будет создана в папке `dist`.

### Предварительный просмотр сборки

```bash
npm run preview
```
